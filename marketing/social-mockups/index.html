📸 Nasıl Ekran Görüntüsü Alınır?

  1. Mockup'ı yeni sekmede aç (kart üzerine tıkla)
  2. Tarayıcı zoom'unu %100 yap (Cmd+0)
  3. Mac: Cmd + Shift + 4 → boşluk → mockup üstüne tıkla (sadece canvas alınır)
  4. Ya da tarayıcı eklentisi: "GoFullPage" veya "Awesome Screenshot" kullan
  5. Daha hızlı: Chrome DevTools → Cmd+Shift+P → "Capture full size screenshot"

🎨 Renk/Yazı Değiştirmek İstersen

  1. HTML dosyasını aç → <style> bloğundan renkleri değiştir
  2. Mock data Türkçe, hepsi gerçekçi isim/ürünle dolu
  3. Logo: brand-logo div'i — kendi SVG'ni koyabilirsin
